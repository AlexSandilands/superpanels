# SPEC — moved

The technical specification has been split by section to keep token usage manageable for any single task.

**Index:** [`docs/spec/README.md`](./docs/spec/README.md)

Each section is its own file under `docs/spec/`, named `NN-topic.md` where `NN` is the section number. Read only the section(s) relevant to your current task.

References elsewhere in the codebase that say `SPEC.md §N` (e.g. `SPEC.md §6.4`) map to `docs/spec/NN-*.md` — the leading number is the section.
